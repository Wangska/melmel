# HikeBook Cebu - Quick Start Guide

## Step 1: Start XAMPP
1. Open XAMPP Control Panel
2. Click **Start** on Apache
3. Click **Start** on MySQL
4. Wait for both to show green "Running" status

## Step 2: Install Database
Go to:
```
http://localhost/hikingphp/INSTALL.php
```
This will:
- Create the database
- Create all tables  
- Insert 9 sample hikes with images
- Create admin account

## Step 3: Access Your App
**Homepage:**
```
http://localhost/hikingphp/
```

**Admin Login:**
```
http://localhost/hikingphp/login.php

Email: admin@hikebook.com
Password: admin123
```

## That's it! You're ready to go! 🎉

### What You Can Do:

**As a Regular User:**
- Browse hikes
- Register an account
- Login
- Book hikes
- View your bookings

**As an Admin:**
- View dashboard with statistics
- Manage all hikes (add, edit, delete)
- Manage all bookings
- Update booking status
- View revenue
